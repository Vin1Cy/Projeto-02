# Projeto-02
Projeto de número 2, aprendizado em Git e Github, AWS e Python. ( Auxílio de IA )

Esse repositório tem como função guardar a construção do meu aprendizado em construção de um Website junta a integração de outras tecnologias como Python no Back-end e serviços AWS ( RDS PostgreSQL)

Esse website é um painel de informações em integração com o banco de dados. Esse banco de dados recebe dados de um jogo do Roblox.
Através desse website, eu consigo manipular os dados de um player do meu jogo do Roblox.
Esse painel tem funções de Admin Hierarchy e Moderation.
